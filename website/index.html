Agri Data.apk သည် မြန်မာနိုင်ငံ စိုက်ပျိုးရေး လယ်ကွင်းဒေတာ စုဆောင်းရန် Flutter mobile app ဖြစ်ပါသည်။ ကျောင်းသားများအတွက် အပင်ရောဂါ၊ ပိုးမွှား၊ အာဟာရ ပုံများကို offline ရိုက်ယူပြီး မိမိဖုန်းတွင် သိမ်းဆည်းကာ ရက်အလိုက် ဒေတာပုံများကို သိမ်းဆည်းနိုင်ပါသည်။ အဆိုပါ ဒေတာများကို app တွင် ZIP ဖိုင်ထုတ်၍ ဆရာထံ ပို့ပေးနိုင်ရန် အတွက် ဒီဇိုင်းပြုလုပ်ထားပါသည်။

Agro_Data-Desktop.exe သည် Desktop App ဖြစ်ပြီး Mobile Phone ရှိ App နှင့် လုပ်ဆောင်ချက်တူညီစွာ လုပ်ဆောင်နိုင်စေရန် ဒီဇိုင်းပြုလုပ်ထားပါသည်။ အဓိကအားဖြင့် IOS အသုံးပြုသော ကျောင်းသားများအနေဖြင့် IOS App မရသေးသည့်အချိန်တွင် Computer ဖြင့် ဒေတာစုဆောင်းမှုများကိုလုပ်ဆောင်နိုင်စေရန် ရည်ရွယ်၍ ဒီဇိုင်းပြုလုပ်ထားခြင်းဖြစ်ပါသည်။

  1. 01 Collect လယ်ကွင်းတွင် ပုံရိုက်စုဆောင်း
  2. 02 Organize ရက်အလိုက် စီစစ် / ZIP ထုတ်
  3. 03 Train AI model လေ့ကျင့်ရန် သုံးမည်
  • Android: Android 7.0 နှင့် အထက်၊ Unknown apps install ခွင့်ပြုရန် လိုနိုင်ပါသည်
  • Android: ပထမအသုံးပြုရာတွင် Storage / All files access ပေးပါ
  • Windows: Agro_Data_Desktop.exe ကို ဒေါင်းလုဒ်လုပ်ပြီး run ပါ (SmartScreen သတိပေးပါက info ကိုနှိပ်ပြီး Run Any Way ခလုပ်ကိုနှိပ်၍ App ကို Run နိုင်ပါသည်။)
